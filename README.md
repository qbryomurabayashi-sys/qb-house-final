# QB House Evaluation Sheet System (Vite Migration)

既存のHTML単一ファイル (`index.html`) から、モダンな **Vite + React + TypeScript** プロジェクトへの移行版です。

## 📁 ディレクトリ構造

```
src/
├── components/          # UIコンポーネント (EvaluationCard, ScoreDashboard等)
├── data/               # 定数データ (評価項目, マスタデータ)
├── types/              # 型定義 (TypeScript interfecs)
├── App.tsx             # メインアプリケーションロジック
├── main.tsx            # エントリーポイント
└── index.css           # グローバルスタイル (Tailwind CSS)
```

## 🛠 技術スタック

*   **Vite**: 高速なビルドツール・開発サーバー
*   **React**: UIライブラリ (v18)
*   **TypeScript**: 静的型付けによる品質向上
*   **Tailwind CSS**: ユーティリティファーストなスタイリング
*   **Recharts**: チャート描画ライブラリ
*   **Lucide React**: アイコンライブラリ

## 🚀 開発の始め方

このプロジェクトを実行するには、Node.js が必要です。

1.  **依存関係のインストール**:
    ```bash
    npm install
    ```

2.  **開発サーバーの起動**:
    ```bash
    npm run dev
    ```
    ブラウザで `http://localhost:5173` を開いてください。

3.  **ビルド (本番用)**:
    ```bash
    npm run build
    ```
    `dist/` フォルダに静的ファイルが生成されます。これをWebサーバーにデプロイするか、ローカルでserveしてください。

## ✨ 主な変更点

*   **コンポーネント化**: 巨大なHTMLファイルを小さな再利用可能な部品（コンポーネント）に分割しました。
*   **型安全性**: TypeScriptの導入により、データ構造の整合性が保証され、バグが混入しにくくなりました。
*   **データ分離**: 評価項目や定数データを `src/data/constants.ts` に分離し、メンテナンス性を向上させました。
*   **モダンな開発体験**: ホットリロードにより、コードを変更すると即座にブラウザに反映されます。

## ⚠️ 注意事項

*   **ローカルストレージ**: ドメイン（URL）が変わるとローカルストレージ（保存データ）は引き継がれません。移行ツールが必要な場合は別途開発が必要です。現在は新しいURLで新規にデータが蓄積されます。
*   **印刷機能**: ブラウザの標準印刷機能を使用しています。A4縦での印刷に最適化されていますが、ブラウザの設定によっては調整が必要な場合があります。
