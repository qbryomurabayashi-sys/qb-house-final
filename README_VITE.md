# 評価入力アプリ (Vite版)

## 環境構築と実行手順

このアプリケーションは最新のWeb技術（Vite, React, TypeScript）を使用しています。
実行するには `Node.js` のインストールが必要です（推奨バージョン: 18以上）。

### 1. 準備

1. **Node.jsの確認**:
   ターミナル（コマンドプロンプトやPowerShell）を開き、以下のコマンドを入力します。
   ```bash
   node -v
   npm -v
   ```
   バージョン番号が表示されればインストール済みです。
   エラーが出る場合は、[Node.js公式サイト](https://nodejs.org/)からインストーラー（LTS版）をダウンロードしてインストールしてください。

2. **依存パッケージのインストール**:
   このフォルダ（`integrated-app`）の中でターミナルを開き、以下を実行します。
   ```bash
   npm install
   ```
   ※ エラーが出る場合、インターネット接続を確認してください。

### 2. アプリの起動

以下のコマンドを実行します。
```bash
npm run dev
```

起動が成功すると、以下のようなURLが表示されます。
```
  Local:   http://localhost:5173/
```
ブラウザでこのURLを開いてください。

### 3. トラブルシューティング

- **「'npm' は内部コマンド...」と表示される / 起動しない**
  Node.jsが正しくインストールされていないか、パスが通っていません。インストール後にPCを再起動してみてください。

- **画面が真っ白になる**
  ブラウザの「開発者ツール(F12)」を開き、「Console」タブを確認してください。エラーメッセージが表示されている場合、その内容を確認してください。

- **スマートフォンのブラウザで確認したい**
  PCとスマホが同じWi-Fiに接続されている状態で、`npm run dev -- --host` を実行してください。表示されるIPアドレス（例: `http://192.168.1.10:5173`) にスマホからアクセスします。

## 注意事項

- このアプリはブラウザの「ローカルストレージ」にデータを保存します。ブラウザのキャッシュをクリアするとデータが消える可能性があります。定期的に「バックアップ」機能（トップページにあります）を使ってJSONファイルを保存してください。
